# Evolution Gaming bootcamp test assignment

## How to run
### Using Intelij Idea:
 - Import project to idea.
 - Run Main

### Using console:
 - Install scala (https://www.scala-lang.org) and sbt (https://www.scala-sbt.org)
 - Navigate to project root folder
 - Type 'run sbt' 
 - To evalute press 'ctrl + d'
